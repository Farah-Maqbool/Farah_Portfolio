Project:

Convert given Website into Code

My instructor gave me a website and asked me to create one similar to it. I used my knowledge and expertise to build the website, and it wasn't easy. However, I continuously worked on it and completed it within four days. A major challenge I faced was making the website responsive. But I tackled this issue by focusing on understanding the original website's design first. Once I grasped the design, I successfully recreated the website.

Key Features:

Making it Responsive:

I wanted the website to look great on any device, so I used media queries to achieve this. It looks great on laptops, phones, and tablets.

Build Structure and Style:

I used HTML as the foundation and used CSS to style it.
